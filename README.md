# 🧠 React Quiz App

A quiz app built while following Jonas Schmedtmann’s React course, then converted to **TypeScript** and refactored using the **Context API** for global state management.

🌐 Live Demo: https://react-quiz-ts-context.netlify.app/

## 🚀 Features

- Multiple-choice quiz questions
- Score tracking system
- Progress indicator
- Start / finish quiz flow
- Global state with Context API
- Fully migrated to TypeScript

## 🛠 Tech Stack

- React
- TypeScript
- Context API
- useReducer
- Vite

## 📌 Purpose

This project was built for learning advanced React state management and TypeScript integration in real applications.
